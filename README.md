# 📧 Email Spam Detector

A Python-based project that detects whether an email is **Spam** or **Not Spam** using a rule-based approach.  
This project demonstrates basic concepts of **Python programming, file handling, text processing, and data analysis**.

---

## 🚀 Features
- Detects spam emails automatically
- Rule-based spam classification
- Simple and easy-to-understand Python implementation
- Demonstrates file handling and text processing
- Beginner-friendly project

---

## 🛠️ Technologies Used
- Python
- File Handling
- JSON Processing
- Basic Data Analysis
- Object-Oriented Programming (OOP)

---

```
EMAIL-SPAM-DETECTOR/
│

└── README.md
```

---

## ⚙️ How It Works
1. The program reads email data from a file.
2. It checks the email content for spam keywords or patterns.
3. Based on predefined rules, the email is classified as:
   - **Spam**
   - **Not Spam**

---

## ▶️ How to Run the Project

1. Clone the repository:
```
git clone https://github.com/sanikakangane/EMAIL-SPAM-DETECTOR.git
```

2. Navigate to the project folder:
```
cd EMAIL-SPAM-DETECTOR
```

3. Run the program:
